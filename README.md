### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/AlexanderWindecker/reactjs-windecker.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the APP
   ```js
   npm start;
   ```
